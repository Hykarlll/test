# TaskMate – Personal Task Manager

## IMS566 Individual Project | UiTM Kampus Puncak Perdana

### Project Description

TaskMate is a modern, responsive task management web application built as a **UI prototype** for the IMS566 course. It demonstrates advanced web design principles, responsive layouts, and data visualisation using Chart.js. All functionality is front‑only – no backend or database – strictly following the UI‑only requirement for the individual assignment.

### Features

- 🔐 **Simulated Authentication** – Hardcoded login credentials (UI only, no real validation)
- 📊 **Interactive Dashboard** – Summary cards + two meaningful charts (task status & priority distribution)
- ✅ **My Tasks** – 9 sample tasks displayed in a clean 3×3 grid with priority badges and due dates
- 🏆 **Completed Tasks** – Table view of finished tasks (3 items)
- 📅 **Calendar View** – June 2025 calendar with task indicators and upcoming deadlines
- 👤 **Profile Page** – User avatar, statistics, editable profile form, and recent activity log
- 📱 **Fully Responsive** – Works seamlessly on desktop, tablet, and mobile devices
- 🎨 **Modern UI** – Dark theme, gradient accents, smooth hover effects, and blur backdrop

### Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5, CSS3, JavaScript | Core structure, styling, and interactivity |
| Tailwind CSS | Utility‑first CSS framework for rapid UI development |
| Chart.js | Data visualisation (doughnut & bar charts) |
| Font Awesome 6 | Icons for navigation, buttons, and visual cues |
| Vanta.js | Animated wave background on the login page |
